Variables of JavaScript

    There are three keywords we can use to assign values in JavaScript:

  • var :

    • used to define a variable globally

    (متغير عام في البرنامج كامل)

  • let :

    • used to define a Variable that is limited in Scope to the current block such as a function or loop

    (متغير محصور في المكان اللذي تم تعريفه فيه علي سبيل المثال في الداله او التكرار)

  • cont :

    • used to define a value that will not change

    (متغير ثابت لا يتغير)
